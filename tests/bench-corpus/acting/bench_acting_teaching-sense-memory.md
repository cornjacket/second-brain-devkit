# Teaching sense memory and relaxation

Sense memory and relaxation are foundational studio exercises, often the first tools a beginning actor learns. Relaxation comes first: teachers guide students to release habitual physical tension, because a tense body chokes off breath, voice, and truthful impulse. Sense memory then trains the imagination to recreate physical sensations — the warmth of the sun, the weight of a cup, a specific smell — vividly enough that the body responds as if they were real. The skill matters because so much acting means reacting to things that are not actually there, from an imaginary meal to biting cold. Coaches keep the work specific and concrete, since a vague sensation produces vague behavior. Mastering these builds the concentration and imaginative belief every later technique depends on, so the actor can fully invest in imaginary circumstances.
