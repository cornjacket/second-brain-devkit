# Obsidian

Obsidian is a local-first Markdown knowledge base whose store, called a vault, is just a folder of plain `.md` files on disk with YAML frontmatter for properties. Notes connect through `[[wikilinks]]` (with autocomplete, aliases, and block or heading references via `^block-id` and `#heading`), and every link is mirrored in the backlinks pane and rendered as an edge in the interactive graph view. A community plugin ecosystem extends the core — Dataview queries turn frontmatter into live tables, Templater scripts new notes, and the Daily Notes and Canvas plugins support journaling and spatial thinking. Because the vault is plain text rather than a proprietary database, it stays portable and future-proof: you can grep it, version it with git, sync it, and edit the same files in any Markdown editor. This file-over-app philosophy makes Obsidian a common substrate for Zettelkasten, PARA, and linking-your-thinking workflows alike.
